# Golden Bagel Cafe

Production-ready static website and serverless foundation for Golden Bagel Cafe in El Cajon, California.

This repository is being built in reviewable phases:

- Next.js static frontend
- Mobile-first brand and ordering UX
- Editable menu data
- Serverless Lambda/API Gateway foundation
- Clover mock/live adapter foundation
- Local SEO and AWS deployment workflow

Full setup, deployment, and security documentation will be completed as part of the implementation phases.
